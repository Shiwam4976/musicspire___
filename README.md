

A modern, single-page static music player(Musicspire) built with HTML/CSS/JavaScript. Features include a playlist sidebar, play/pause/prev/next controls, progress/seek, volume & playback rate, a canvas visualizer (Web Audio API), drag-and-drop playlist reordering, keyboard shortcuts, theme toggle, and localStorage persistence for settings.



---

 Deployment

This project includes a `vercel.json` for deploying as a static site on Vercel. Steps to deploy are in `README-VERCEL.md` (also included in the repo).


---



- Playlist built from local `Music/` and `images/` assets.
- Clickable playlist with track cover thumbnails.
- Play / Pause / Previous / Next / Shuffle / Repeat.
- Click-to-seek progress bar and time display.
- Volume control and playback speed selector.
- Canvas visualizer using the Web Audio API.
- Drag & drop reordering of the playlist (in-memory; can be persisted to localStorage).
- Keyboard shortcuts: Space — Play/Pause, Left/Right arrows — Prev/Next.
- Dark theme toggle and persistence for user settings.
